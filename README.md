## ✨ Features

* Remaining **jump count**
* Display of **star classes**
* Highlighting of **scoopable stars**
* Preview of the **next 5 systems**
* **Always-on-top overlay**
* Freely movable window
* **Click-through support**
* Context menu (right-click)
* Adjustable:

  * Font size
  * Background transparency
* Exit via:

  * Double-click
  * Right-click
    Source: https://forums.frontier.co.uk/threads/scoopable-stars-jump-counter.646101/

---

## 🖥️ Requirements

* Windows
* **.NET 8 Runtime**

---

## 🚀 Installation

1. Download latest release:
   https://github.com/UncleRiot/EDtoolset/releases
2. Run `EDtoolset.exe`
3. Start Elite Dangerous

---

## ⚙️ Usage

* Overlay automatically shows:

  * remaining jumps
  * star types along route
* **Right-click** → settings
* **Drag & drop** → reposition
* **Double-click** → exit
  Source: https://forum.stellanebula.de/viewtopic.php?t=1530
